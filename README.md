# bbdd-ecommerce

Descripción: Proyecto final de BBDD para una plataforma ficticia e-commerce de alquiler de ropa.

Tecnologías: MySQL Workbench & XAMPP.

Contenido: El repositorio incluye el diagrama ER, los scripts SQL para crear y probar la base de datos y un archivo con +30 consultas complejas (JOINS, subconsultas).
